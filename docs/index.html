核心能力

不只是写代码，是工作流思维

CCclaw 不是另一个代码补全工具。它拥有完整的编程工作流——先理解、再规划、后执行。

🧠

Plan Mode — 先规划再执行

3 步以上的任务自动进入规划模式。AI 先读代码、理解架构、输出方案，等你确认后再动手。简单修复直接跳过，不啰嗦。

你: "给这个接口加重试机制"

传统 AI: 直接开始改代码……改了一半发现改错了 😅
CCclaw: 1. 读代码 → 2. 理解架构 → 3. 出方案
         4. 等你确认 ✅ → 5. 再动手
🔒

Git 安全协议

禁止 force push、reset hard、--no-verify。Commit 前必须先检查 status + diff + log。

📋

任务追踪

多步任务自动创建清单，实时更新状态。同一时间只有一个「进行中」，卡住了自动标阻塞。

🎯

LSP 语言服务器

不是 grep 瞎搜，是精准代码导航。用 TypeScript Compiler API 实现，8 种操作覆盖所有代码理解场景。

goto 定义 查找引用 悬停签名 符号搜索 实现跳转 调入层级 调出层级 全文搜索
🔍

只读探索

理解代码时只读不改。避免「看看代码 → 手滑改了 → 改坏了」的经典翻车。

📓

Notebook 编辑

直接读写 .ipynb 文件，增删改 cell。做数据分析不用手动切 Jupyter。

💾

上下文管理

会话记忆 · 智能压缩 · 记忆提取。长对话不丢上下文，自动从对话中保留关键信息。

💬

多渠道接入

微信、飞书、QQ、Discord、Telegram、Slack、Signal……在任何地方都能和 AI 编程。

🔗

Brief 附件

任务生成的文件自动附带发送。图片、日志、报告——不用手动找路径。
