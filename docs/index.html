CCClaw

让 AI 像专业程序员一样思考和工作

⬇️ 下载桌面版 GitHub

核心能力

不只是「能写代码」，而是拥有完整的编程工作流思维。

🧠

Plan Mode — 先规划再执行

3 步以上的任务自动进入规划模式。AI 先读代码、理解架构、输出方案，等你确认后再动手。简单修复直接跳过，不啰嗦。

你: "给这个接口加重试机制"

传统 AI: 直接开始改代码
CCclaw: 读代码 → 理解架构 → 出方案 → 等确认 → 再动手
📋

任务追踪

多步任务自动创建清单，实时更新状态。同一时间只有一个「进行中」，卡住了自动标阻塞。

✅ 分析现有路由结构
✅ 设计重试策略
🔄 修改 middleware
⏳ 更新调用方
⏳ 跑测试
🔒

Git 安全协议

禁止危险操作：force push、reset hard、--no-verify。Commit 前必须先检查 status + diff + log。

git push --force → ❌ 已禁用
🔍

只读探索模式

理解代码时 AI 只能读不能改。避免「看看代码 → 手滑改了 → 改坏了」的经典翻车。

🎯

LSP 语言服务器

不是 grep 瞎搜，是精准代码导航。用 TypeScript Compiler API 实现，支持 8 种操作：

goto 定义 引用 悬停 符号搜索 实现 调入 调出 搜索
📓

Notebook 编辑

直接读写 .ipynb 文件，支持 cell 的增删改。做数据分析不用手动切 Jupyter。

replace / insert / delete cell
🔗

Brief 附件

完成任务后生成的文件（图片、日志、报告）自动附带发送，不用手动查找路径。

💾

上下文管理

会话记忆 — 结构化模板，长对话不丢上下文
智能压缩 — 对话太长时保留关键信息，丢弃冗余探索过程
记忆提取 — 自动从对话中提取值得长期保存的信息

💬

多渠道接入

不只 Webchat。支持微信、飞书、QQ、Discord、Telegram、Slack、Signal 等 10+ 渠道。在任何地方都能和 AI 编程。

和现有方案的区别

CCclaw 不是另一个 Copilot。它是有工作流思维的编程助手。
